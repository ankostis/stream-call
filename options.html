About Stream call

A (mobile) Firefox extension to extract streaming media URLs (podcasts, radio stations, live streams) and send to HTTP API endpoint(s).

Supported: HLS (.m3u8), DASH (.mpd), HTTP audio, RTMP/RTSP, Icecast/Shoutcast.
