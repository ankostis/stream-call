Note: pageUrl and pageTitle are only included if the checkbox above is enabled.

About Stream call

Stream call detects streaming media URLs (podcasts, radio stations, live streams) on web pages and allows you to send them to your configured HTTP API endpoint.

Supported stream types: HLS (.m3u8), DASH (.mpd), HTTP Audio (MP3, AAC, OGG), RTMP, RTSP, Icecast, Shoutcast, and more.
